Template Generator → Sterling Designer Integration Demo

A hosted proof of concept showing an AI-generated design being pushed into a Sterling-format product designer as fully editable objects, with automatic product recommendations and safe duplication for other products.

This is not Sterling's production designer. It is a browser-only test harness using the same canvas engine (Fabric.js 4.4.0) and the same template data format (the "version 1.2" envelope). It contains demonstration products only (DEMO- numbers), no customer data, no cart, no ordering, no uploads, and it makes no connection to any Sterling server or database. A safety guard blocks and logs any attempted request to Sterling production domains.

New: the demo now also includes a static test copy of the real Sterling designer interface — the genuine browser code customers use on sterling.ca (Fabric.js 4.4.0 + SMPdesigner), running against embedded test fixtures instead of Sterling's servers. A pushed design defines its own product size, so business cards, signs, posters, and other dimensions all open at the correct canvas size. Saving, carts, uploads, and proofs are disabled; no request leaves the page to any Sterling system. Push to Designer sends your design straight into this real interface, and a "View product recommendations" button links onward to the compatibility screen. Solid shapes (bands, wedges, circles) now arrive as native editable vector shapes; gradients, shadows and textures are flattened at high resolution behind them (see docs/TIER1_NATIVE_SHAPES.md). Sterling's fonts aren't bundled in this copy, so text falls back to system fonts. The generator's "Demo shortcuts" now include several real AI-generated cards to try with no API key.

How to test it

  1. Click Launch Integration Demo to open the test Design Template Generator.
  2. Click one of the "Load sample design" demo shortcuts (no API key needed) — or generate a real design with your own Anthropic API key.
  3. Click the orange Push to Designer button in the preview toolbar.
  4. The real Sterling designer interface (test copy) opens with your design as editable objects — click any text to select it, double-click to edit, exactly as customers would.
  5. Click "View product recommendations" (bottom-right) to see the design's detected size and demonstration products grouped by compatibility, with reasons and warnings.
  6. Click "Duplicate design for this product" on a same-size product, then on a different-size one — each copy appears as a new tab, scaled and centred, while the original tab stays untouched.

What this demo proves

  • Generated designs convert to Sterling-compatible version 1.2 template JSON.
  • Text, shapes, and images remain editable after transfer.
  • Design dimensions, orientation, and aspect ratio are detected automatically.
  • Compatible products are recommended with plain-language reasons; incompatible ones stay selectable with warnings.
  • Duplicating for another product never modifies the original design.

What still needs Sterling servers
